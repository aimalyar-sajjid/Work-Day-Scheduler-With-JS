# Work Day Scheduler

## Description:
This app can help you to save your daily schedule and do your jobs according to your schedule, It can tells you about the past, present and future tasks, you added to your scheduler, You can easily save and set your scheduler, Your saved task will not remove after refreshing the browser, because all staffs save in local storage.


## Technology:
This beautiful project created using:
- HTML 5
- CSS 3
- JavaScript



## Usage
Get the code and run 'index.html' file on your browser.
